# Card 005: Class Material Image Assets

## Status

Accepted.

## Intent

The Italian Learning Workbench should capture images that correspond with actual class material and turn them into speaking aids, corpus entries, and practice loops.

## Why

Class slides and generated images are useful when they preserve a real learning situation. A picture of `Come si chiama?` is not merely an illustration. It can become a reusable prompt for asking names, giving a first-person answer, naming things, repairing misunderstanding, and capturing personal corpus entries.

## Core rule

Images that correspond with class material must stay attached to:

- source image path
- class phrase family
- communicative need
- first-person learner line
- call-and-response practice
- pronunciation note
- repair phrase
- personal corpus capture

## Practice loop

```text
Hear → Imitate → Answer → Repair → Capture
```

## Governance

The image is not a decorative artifact. It is a class-corresponding corpus asset. Future images should be added only when they map to a phrase family, a situation, and a speaking loop.
