# Card 014 — Add Class 1 screenshot phrase flashcards

## Purpose

Turn the collected Class 1 Zoom/screenshot notes into first-class flashcards rather than leaving them only as images.

## Accepted behavior

- The site uses the CB013 category filter UI.
- Flashcard categories now match the Class 1 notes actually collected: saluti, presentazioni, come-stai, nome, essere, stare, tu-lei, formale, informale, riparazione, frasi-utili, dialogo, sostantivi, verbi, frasi, domande, risposte, curated, needs-image.
- New FCs are added for name exchange, the formal Mark/employee dialogue, Tu o Lei, Come stai responses, and the informal dialogue model.
- Each FC has `italian`, `speak`, `image`, `curated`, and `categories` metadata.
- Class note screenshots are copied into `site/images/vocabulary/class-notes/` for website display and into `assets/class_material/first_class/` for Workbench custody.

## Notes

Corrected obvious slide typos in the spoken/card text: `valige` becomes `valigie`, `Dova abita?` becomes `Dove abita?`, and `Impietago` is treated as `Impiegato`.
