# Card 009 — Classroom Vocabulary Curation Contract

The Italian vocabulary site maintains a classroom-derived vocabulary ledger.

Tabs:
- Nowns
- Verbs
- Other

Each entry records the Italian term, English gloss, classroom source, curated flag, website image reference, and image-generation prompt/essence.

The curated flag gates image generation:
- `curated: false` = captured, not image-ready
- `curated: true` = reviewed and ready for LLM image generation/refinement

This keeps the site aligned with actual class learning rather than generic vocabulary expansion.
