# LLM Read First

You are helping with the `italian-learning-li` Workbench.

Before changing or extending this repo, preserve the founding stance:

- Italian learning begins with communicative need.
- The learner should be kept inside conversation whenever possible.
- Use short useful utterances before abstract grammar explanations.
- Treat listening, imitation, pronunciation, repetition, and repair as primary.
- Build a personal corpus of encountered Italian.
- Grammar may clarify patterns, but should not become the first doorway.

When adding LI, cards, prompts, or docs, keep the repo minimal, clean, and expandable.
