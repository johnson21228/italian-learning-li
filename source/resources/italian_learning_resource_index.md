# Italian Learning Resource Index

This index registers uploaded Italian learning resources so future WB conversations can find their value quickly.

## Standard location

```text
source/resources/
```

## Resource custody

Governed by:

```text
li/source/uploaded_resource_custody_rule.md
```

The index preserves summaries and use maps. It does not require the repository to store complete source PDFs.

## Registered resources

### 1. Piacere!

```text
source/resources/piacere_resource_summary.md
```

**Uploaded file name:** `Piacere-1669134974._print.pdf`  
**Role:** modern elementary Italian textbook / structured curriculum support.  
**High-value use:** align class practice with a full beginner sequence, especially `Saluti e presentazioni`, alphabet, greetings, introductions, numbers, pronouns, `avere`, `essere`, vocabulary, conversations, readings, and exercises.  
**Best WB use:** curriculum map, conversation prompts, grammar-after-use clarification, vocabulary families, personal corpus expansion.

### 2. Cortina Conversational Italian in 20 Lessons

```text
source/resources/cortina_conversational_italian_resource_summary.md
```

**Uploaded file name:** `Cortina Conversational Italian in 20 Lessons.pdf`  
**Role:** older conversation-oriented self-study course.  
**High-value use:** conversational lesson framing, phonetic-pronunciation support, and an old-school lesson progression that may be mined carefully for practice patterns.  
**Best WB use:** reference model for self-study drills and pronunciation support. Do not copy long passages into repo outputs.

### 3. Basic Italian for Travelers

```text
source/resources/basic_italian_traveler_resource_summary.md
```

**Uploaded file name:** `Basic Italian Traveler Text.pdf`  
**Role:** practical traveler phrasebook.  
**High-value use:** immediate need-to-speak categories: greetings, etiquette, introductions, numbers, food, taxis, hotels, communication strategies, shopping, emergencies, telephone etiquette, signs, geography, and cultural facts.  
**Best WB use:** quick phrase families, travel scenario roleplay, pronunciation drills, signs and survival communication.

## How to use this index in future conversations

Before inventing a new lesson, ask:

1. Does the learner need structured beginner curriculum? Use `Piacere!`.
2. Does the learner need old-school conversational drill structure? Use `Cortina`.
3. Does the learner need immediate travel survival speech? Use `Basic Italian for Travelers`.
4. Can the material become a small first-person exchange?
5. What should be captured in the personal corpus?
