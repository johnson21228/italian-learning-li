# Hear → Imitate → Answer → Repair → Capture

This is the first daily practice loop for the Italian Learning Workbench.

## Loop

### 1. Hear

Start with one useful Italian phrase in a real situation.

Example:

```text
Come ti chiami?
```

### 2. Imitate

Repeat the phrase aloud. Keep it short. Prefer a living spoken phrase over a rule explanation.

### 3. Answer

Give a small first-person answer.

```text
Mi chiamo Steve.
```

### 4. Repair

When the learner does not understand, use repair language immediately:

- Non capisco.
- Puoi ripetere?
- Più lentamente, per favore.
- Che significa?
- Come si dice in italiano?

Repair is not failure. Repair is part of conversation.

### 5. Capture

Add a personal corpus entry.

A corpus entry captures:

- phrase heard
- phrase spoken
- situation
- meaning
- pronunciation note
- repair needed
- next phrase wanted

## Practice posture

The learner should not leave the conversation to study a chapter. The learner should use the phrase, repair the exchange, and then optionally ask what pattern appeared.
