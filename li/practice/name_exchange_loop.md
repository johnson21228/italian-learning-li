# Name Exchange Loop

## Purpose

Practice introducing oneself and asking for another person's name.

## Informal exchange

```text
A: Come ti chiami?
B: Mi chiamo Steve.
```

Alternate response:

```text
B: Sono Steve.
```

## Formal exchange

```text
A: Come si chiama?
B: Sono il signore Johnson.
```

```text
A: Come si chiama?
B: Sono la signora Johnson.
```

## Pronunciation focus

`chiami` and `chiamo` use a hard k sound:

```text
chiami → kee-AH-mee
chiamo → kee-AH-mo
```

## Loop

1. Hear the question.
2. Repeat the question.
3. Answer with `Mi chiamo ___` or `Sono ___`.
4. Repair if needed with `Puoi ripetere?` or `Più lentamente, per favore.`
5. Capture the useful phrase in the personal corpus.

## Constraint

Do not start with reflexive verb explanation. First make the exchange work.
