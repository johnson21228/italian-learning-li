# Childlike Language Acquisition Rule

## Rule

Approach Italian like a child learning to communicate before learning to analyze grammar.

## Meaning

Start with attention, desire, social contact, imitation, repetition, pointing, naming, and repair.

The learner should practice speech that is close to ordinary life:

- `questo` — this
- `quello` — that
- `voglio` — I want
- `ho bisogno di` — I need
- `mi piace` — I like
- `Non capisco.` / `non capisco` — I do not understand
- `ripeti, per favore` — repeat, please

## Practice shape

A good exercise is small and bodily:

1. Look at something.
2. Ask what it is called.
3. Hear the answer.
4. Repeat it.
5. Use it in a tiny sentence.
6. Ask again if confused.

## Guardrail

Do not confuse childlike acquisition with childish content. The learner is an adult. The method is direct, embodied, repetitive, social, and meaning-first.
