# Italian Learning Principles

## First principle

Italian learning begins with the basic human drive to communicate.

The learner should be treated as a person trying to say something, not as a student trying to pass a grammar quiz.

## Primary learning acts

The earliest Italian should serve immediate conversational purposes:

- greeting: `Ciao.` / `Buongiorno.`
- naming: `Mi chiamo Steve.`
- asking: `Come si chiama?`
- wanting: `Voglio un caffè.`
- needing: `Ho bisogno di aiuto.`
- noticing: `Questo è...`
- answering: `Sì.` / `No.` / `Va bene.`
- repairing: `Non capisco.` / `Puoi ripetere?`

## Learning order

Prefer:

1. hear it;
2. repeat it;
3. use it in a tiny situation;
4. repair it when it fails;
5. notice a pattern;
6. name the grammar only when it helps.

Do not lead with abstract grammar when a phrase can be learned as useful speech.

## Personal corpus

The Workbench should gradually build a personal corpus of lived Italian:

- phrase heard;
- phrase spoken;
- situation;
- meaning;
- pronunciation note;
- repair phrase;
- next phrase needed.

The corpus is not just vocabulary. It is a record of attempts to communicate.
