# Class 1 Note Flashcard Category Rule

For the Class 1 flashcard site, categories should match the class notes that were actually collected before adding broader language-instruction taxonomy.

## Controlled Class 1 categories

Use this controlled category vocabulary unless a later CB explicitly expands it:

- `class-1`
- `saluti`
- `presentazioni`
- `come-stai`
- `nome`
- `essere`
- `stare`
- `tu-lei`
- `formale`
- `informale`
- `riparazione`
- `frasi-utili`
- `dialogo`
- `sostantivi`
- `verbi`
- `frasi`
- `domande`
- `risposte`
- `curated`
- `needs-image`

## Required FC metadata

Every flashcard must include:

- `id`
- `italian`
- `english`
- `speak`
- `image`
- `imageAlt`
- `partOfSpeech`
- `categories`
- `curated`

The `speak` field is the exact Italian text the speech button should say.

## Class note source images

Screenshots used as flashcard images must be preserved in both places:

- `site/images/vocabulary/class-notes/` for the web app
- `assets/class_material/first_class/` for Workbench custody

## Category discipline

Do not invent near-duplicate categories like `name-exchange`, `introductions`, or `formal-you` while the Class 1 note category set is active. Use the canonical Class 1 category names above.
