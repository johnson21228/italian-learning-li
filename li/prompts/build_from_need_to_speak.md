# Prompt: Build From Need To Speak

Use this prompt when the learner has an English desire, need, or situation and wants the simplest useful Italian.

```text
Take this English need-to-speak sentence and turn it into the simplest useful Italian utterance.

Do not start with grammar.
Start with what I can say in the moment.
Give me:
1. the Italian phrase;
2. a slow pronunciation guide;
3. a tiny call-and-response practice;
4. one repair phrase if I am not understood.

English need: <write the English here>
```

## Examples

```text
English need: I want coffee.
```

Should produce something centered on:

```text
Voglio un caffè.
```

```text
English need: I don't understand.
```

Should produce something centered on:

```text
Non capisco.
Puoi ripetere lentamente?
```
