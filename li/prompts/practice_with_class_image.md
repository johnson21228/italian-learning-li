# Prompt: Practice With Class Image

Use this prompt when practicing from a class-corresponding image asset.

```text
Use the Italian Learning LI and the class image corpus entry.

Image/corpus focus:
li/corpus/come_si_chiama_image_corpus_entry.md

Keep me inside conversation. Ask one Italian question at a time. Use this loop:
Hear → Imitate → Answer → Repair → Capture

Start with:
Come ti chiami?

Wait for my answer. If I answer in English, help me convert it into the simplest Italian utterance. If I get stuck, offer:
Mi chiamo Steve.

Then ask:
Come si chiama questo?

Use repair phrases when needed:
- Non capisco.
- Puoi ripetere lentamente?
- Come si dice in italiano?

At the end, create one personal corpus entry with:
- phrase heard
- phrase spoken
- situation
- meaning
- pronunciation note
- repair needed
- next phrase wanted
```
