# Prompt: Create Image From Class Material

Use this prompt when creating a learning image that corresponds to actual Italian class material.

```text
Create an Italian learning image from this class material.

Do not make a generic poster. Use the exact phrase family from class and turn it into a visual speaking aid.

Class phrase family:
- Come ti chiami?
- Mi chiamo Steve.
- Come si chiama questo?
- Si chiama ___
- Non capisco. Puoi ripetere lentamente?

Design goal:
The image should help the learner speak, not just read. It should support a tiny conversation with one question, one answer, and one repair phrase.

Required visual sections:
1. Phrase to hear
2. Phrase to repeat
3. First-person answer
4. Object/person naming example
5. Repair phrase
6. Personal corpus capture cue

Use Italian-first text. Keep English out of the image unless explicitly requested.

The image must support this loop:
Hear → Imitate → Answer → Repair → Capture
```
