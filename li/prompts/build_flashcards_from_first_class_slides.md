# Prompt — Build Flashcards From First Class Slides

Use this prompt to create practice cards from the first class corpus.

```text
Build a small flashcard deck from the first class Italian corpus.

Make cards for:
- greetings
- leave-taking
- Come stai? / Come sta? / Come va?
- response choices
- Come ti chiami? / Mi chiamo ___
- Come si chiama?
- Tu / Lei / voi
- first exposure to essere and stare

Each card should include:
- front
- back
- situation
- formal/informal note when relevant
- one spoken example

Do not over-explain grammar. Keep the deck useful for speaking.
```
