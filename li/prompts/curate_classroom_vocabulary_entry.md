# Curate Classroom Vocabulary Entry

Use this prompt when adding or refining vocabulary introduced in class.

Goal:
Maintain the classroom vocabulary ledger for the Italian learning site.

For each term:

1. Confirm the word was introduced by class material, class notes, class image discussion, or derived classroom practice.
2. Classify it into exactly one tab:
   - Nowns
   - Verbs
   - Other
3. Add or update:
   - Italian term
   - English gloss
   - source or classroom context
   - curated flag
   - image essence
   - image prompt
   - website image path or placeholder
4. Set `curated: false` when the entry is only captured.
5. Set `curated: true` only when the term, meaning, tab, and image essence are reviewed.

When curated is true, the entry is ready for an LLM image-generation pass.
