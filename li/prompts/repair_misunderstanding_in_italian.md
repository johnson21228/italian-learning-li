# Prompt: Repair Misunderstanding In Italian

Use this prompt when the learner gets stuck or does not understand.

```text
Create a short Italian repair loop for misunderstanding.

Teach and practice these phrases:
- Non capisco.
- Puoi ripetere?
- Più lentamente, per favore.
- Che significa?
- Come si dice in italiano?

Do not turn this into a grammar lesson.
Use call-and-response.
Say each phrase once normally, once slowly, and then ask me to answer with it in a tiny conversation.

End by asking which phrase I want to keep in my personal corpus.
```
