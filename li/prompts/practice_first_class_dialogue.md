# Prompt — Practice First Class Dialogue

Use this prompt to turn the first class slide material into a live micro-conversation.

```text
Using the Italian Learning LI rules, practice the first class dialogue with me.

Use only beginner Italian from the first class corpus:
- Ciao
- Buon giorno
- Buona sera
- Come stai?
- Come sta?
- Come va?
- Sto bene
- Così così
- Non c'è male
- Come ti chiami?
- Mi chiamo ___
- A presto
- Arrivederci

Keep me inside conversation. Ask me one question at a time. Wait for my answer. Correct gently. Use English only when needed to keep me from getting lost.

After the exchange, make one personal corpus entry for the phrase I most need next.
```
