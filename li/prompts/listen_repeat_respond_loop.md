# Prompt: Listen, Repeat, Respond Loop

Use this prompt to practice hearing and speaking before analysis.

```text
Run a listen-repeat-respond loop for this Italian phrase:

Phrase: <Italian phrase>

Do this:
1. say the phrase normally;
2. break it into small sound chunks;
3. give an English meaning;
4. ask me to repeat it;
5. give me a tiny situation where I would say it;
6. ask me a yes/no or either/or question so I can respond in Italian;
7. include one repair phrase: Non capisco. Puoi ripetere lentamente?
```

## Good starter phrases

```text
Come si chiama?
Mi chiamo Steve.
Voglio un caffè.
Non capisco.
Puoi ripetere lentamente?
Come si dice in italiano?
```
