# Prompt: Start Italian Micro-Conversation

Use this prompt to begin a tiny Italian conversation.

```text
Start a very short Italian micro-conversation with me.

Keep me inside the conversation as a first-person speaker.
Use only a few lines.
Include pronunciation help.
Make me repeat after you.
Then ask me one simple question in Italian.

Focus on: greeting, name, and asking what something is called.
```

## Expected behavior

The response should avoid a long lesson. It should create a small speaking loop, such as:

```text
Ciao.
Mi chiamo Steve.
Come si chiama?
Si chiama...
```
