# Prompt — Use Uploaded Resource for Practice

Use the Italian Learning LI and the uploaded resource index.

Start here:

```text
source/resources/italian_learning_resource_index.md
li/source/uploaded_resource_custody_rule.md
```

Choose the best resource for the learner's current need:

- `Piacere!` for structured beginner course alignment;
- `Cortina Conversational Italian in 20 Lessons` for old-school conversational drill patterns and pronunciation support;
- `Basic Italian for Travelers` for immediate survival/travel speech.

Do not give a long textbook explanation first.

Convert the resource into a small practice loop:

```text
Hear → Imitate → Answer → Repair → Capture
```

Output format:

1. one tiny Italian exchange;
2. pronunciation help for one phrase;
3. one learner response prompt;
4. one repair option;
5. one personal corpus entry.

Keep Italian first. Use English only briefly when needed.
